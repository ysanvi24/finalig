# 🎯 QUICK START FOR TEAMMATES

## ⚡ Get Started in 3 Commands

```bash
git clone https://github.com/Anshulkaocde123/finalig.git
cd finalig
chmod +x setup-team.sh && ./setup-team.sh
```

Then start the app:
```bash
npm start
```

**App URLs:**
- Frontend: http://localhost:5173
- Backend: http://localhost:5000
